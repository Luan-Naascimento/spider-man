# Spider-Man CSS Project 🕷️

Projeto simples desenvolvido para praticar **estilização com CSS** e testar conceitos de layout e design em páginas web.

## 📌 Objetivo

O objetivo deste projeto foi realizar experimentos com **CSS**, explorando estilos visuais, posicionamento de elementos e personalização de interface utilizando HTML e CSS.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3

## ⚙️ Funcionalidades

- Página web simples com tema do Spider-Man
- Testes de estilização com CSS
- Aplicação de cores, posicionamento e organização de elementos

## 📂 Estrutura do projeto

```
spider-man
│
├── index.html
├── style.css
└── assets
```

## 🧠 Aprendizados

Durante o desenvolvimento deste projeto foram praticados:

- Estilização de páginas com CSS
- Organização de layout
- Estrutura básica de páginas HTML
- Experimentação com design visual

## 📚 Contexto

Este projeto foi desenvolvido como **experimento de aprendizado**, com o objetivo de testar e melhorar habilidades em **CSS**.
